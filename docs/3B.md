## Files Created / Modified

### 1. Database Migration
- [supabase_migration_3b.sql](file:///a:/June2026/happy-soul/supabase_migration_3b.sql)
  - Adds the `is_onboarded` boolean field (defaulting to `false`) to the existing `profiles` table.

### 2. Route Guard Protection
- [middleware.js](file:///a:/June2026/happy-soul/lib/supabase/middleware.js)
  - Checks `profiles.is_onboarded` for authenticated users.
  - Redirects users with `is_onboarded = false` to `/onboarding` if they try to browse other pages.
  - Redirects users with `is_onboarded = true` away from `/onboarding` to `/dashboard`.

  - Top visual circles indicating step completion.
- [ProgressBar.jsx](file:///a:/June2026/happy-soul/components/onboarding/ProgressBar.jsx)
  - Uses Framer Motion to animate progress smoothly as steps transition.
- [StepCard.jsx](file:///a:/June2026/happy-soul/components/onboarding/StepCard.jsx)
