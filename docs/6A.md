## Files Created / Modified

### 1. Database Migration Script
* [supabase_migration_6a2.sql](file:///a:/June2026/happy-soul/supabase_migration_6a2.sql):
  * **seed_metadata**: Table storing import records (dataset_name, translator, commentary, version, counts, checksum, seeded_at).
  * **RLS & Policies**: Row Level Security enabled. Read-only SELECT policy allowed for `authenticated` users.

### 2. Modular Seeding Engine (`scripts/gita/`)
* [fetch.js](file:///a:/June2026/happy-soul/scripts/gita/fetch.js): Implements `fetchWithRetry` doing 3 attempts with exponential backoff on HTTP/network issues.
* [utils.js](file:///a:/June2026/happy-soul/scripts/gita/utils.js): Manually parses `.env.local` to process credentials and uses `crypto` for content SHA-256 hashes.
* [seedChapters.js](file:///a:/June2026/happy-soul/scripts/gita/seedChapters.js): Stage 1. Checks received JSON array and maps/upserts 18 records into `public.gita_chapters`. Returns `chapter_number -> UUID` map.
* [seedVerses.js](file:///a:/June2026/happy-soul/scripts/gita/seedVerses.js): Stage 2. Maps and inserts 700 verses linked to chapter UUIDs in batches of 50.
* [enrichTranslations.js](file:///a:/June2026/happy-soul/scripts/gita/enrichTranslations.js): Stage 3. Resolves English translations (filtering by `DEFAULT_TRANSLATOR` configured in `lib/gita/config.js` with language fallback) and updates `gita_verses`.
* [enrichCommentaries.js](file:///a:/June2026/happy-soul/scripts/gita/enrichCommentaries.js): Stage 4. Resolves commentaries (filtering by `DEFAULT_COMMENTARY` configured in `lib/gita/config.js` with language fallback) and updates `gita_verses`.
* [index.js](file:///a:/June2026/happy-soul/scripts/gita/index.js): Coordinates the stages. Calculates dataset checksum of all downloaded JSON strings, runs the seeder pipeline, and logs the execution to `public.seed_metadata`.

### 3. Package Configurations
* [package.json](file:///a:/June2026/happy-soul/package.json): Added `"db:seed-gita": "node scripts/gita/index.js"` script target.

### 4. Seeding Documentation
* [gita-seeding.md](file:///a:/June2026/happy-soul/docs/gita-seeding.md): Clear instructions detailing Supabase backup advice, environment keys, running scripts, and validating record counts.
